## Usage

**Roll**:
`!roll 1d6` for one 6-sided dice 

`!roll 6x4d6` for six sets of four six-sided dice

`!roll 2dF` or `!rollfate 2` to roll 2 "fate dice" (six-sided dice with sides labelled +, -, and 0)
